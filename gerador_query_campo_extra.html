Gerador de Queries SQL para Campos Extras

Preencha o formulário abaixo para gerar automaticamente a query SQL de inclusão de campo extra.

Campos Identificadores

? Este é o texto que será exibido para o usuário no formulário. Exemplo: "Nome Completo", "E-mail", "Telefone".
? Este código é gerado automaticamente a partir do nome do campo. É usado internamente pelo sistema.

Módulo

? Selecione o módulo do sistema onde este campo deve ser exibido. A seção onde o campo aparece depende do módulo selecionado.

Campos de Configuração

? Escolha em qual seção do formulário este campo será exibido. Se não tiver certeza, use "Informações Adicionais".
? Escolha o tipo de entrada que o usuário irá preencher. Texto curto para respostas breves, texto longo para descrições, data para datas, etc.
? Número que define a ordem em que este campo aparecerá no formulário. Campos com números menores aparecem primeiro.

Permissões

? Define o que administradores podem fazer com este campo:
- NENHUMA: não podem ver nem editar
- LEITURA: podem apenas visualizar
- ESCRITA: podem visualizar e editar
? Define o que usuários comuns podem fazer com este campo:
- NENHUMA: não podem ver nem editar
- LEITURA: podem apenas visualizar
- ESCRITA: podem visualizar e editar

Configurações Adicionais

Marque as opções desejadas para este campo:

? Se marcado, este campo será exibido na loja virtual (e-commerce).
? Se marcado, o usuário será obrigado a preencher este campo.
? Se marcado, este campo poderá ser usado como filtro em pesquisas.
? Se marcado, este campo será exibido nos relatórios gerados pelo sistema.
? Se marcado, este campo será considerado um campo padrão do sistema.
? Se marcado, o sistema não permitirá valores repetidos neste campo.
? Se marcado, este campo poderá ser incluído em certificados gerados pelo sistema.
? Se marcado, o valor deste campo poderá ser incluído em mensagens automáticas.
? Se marcado, o sistema forçará o usuário a editar este campo em determinadas situações.
? Se marcado, este campo será exibido em listas de presença.

Campos Descritivos

Campos opcionais. Deixe em branco se não for necessário.

? Define um formato específico para o campo. Exemplo: (99) 9999-9999 para telefone, onde 9 representa um dígito.
? Texto de exemplo que aparece no campo antes do usuário digitar. Exemplo: "Digite seu email".
? Valor que aparecerá preenchido automaticamente no campo. Exemplo: "Selecione uma opção".

Opções de Seleção

Adicione as opções que o usuário poderá selecionar.

? Para campos do tipo BOTÕES DE OPÇÃO, LISTA SUSPENSA, CHECKBOX, etc., você precisa adicionar as opções que o usuário poderá escolher. Exemplo: "Sim", "Não", "Talvez".
